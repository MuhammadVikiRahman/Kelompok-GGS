5 Elemen HTML Dasar

Lima elemen HTML dasar yang menjadi fondasi dalam struktur setiap halaman web.

1

<html>

Elemen <html> adalah elemen root yang membungkus semua konten HTML pada halaman. Ini adalah elemen container level tertinggi yang menampung seluruh struktur dokumen HTML.

Semua elemen HTML lainnya harus menjadi turunan dari elemen ini. Elemen ini juga dikenal sebagai root element. 

<!DOCTYPE html>
<html lang="id">
  <head>
    <!-- Konten head -->
  </head>
  <body>
    <!-- Konten body -->
  </body>
</html>
<html>
<head> ... </head>
<body> ... </body>
</html>
2

<head>

Elemen <head> berisi metadata dan informasi tentang dokumen HTML, seperti judul, karakter set, stylesheet, dan script.

Konten dalam elemen head tidak ditampilkan di browser, tetapi berisi informasi penting tentang dokumen. 

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman</title>
  <link rel="stylesheet" href="styles.css">
  <script src="script.js"></script>
</head>

3

<body>

Elemen <body> berisi seluruh konten yang akan ditampilkan di browser, seperti teks, gambar, link, tabel, dan elemen HTML lainnya.

Semua konten yang terlihat oleh pengguna harus ditempatkan di dalam elemen body. 

<body>
  <header>
    <h1>Judul Website</h1>
    <nav>
      <!-- Menu navigasi -->
    </nav>
  </header>
  
  <main>
    <p>Ini adalah paragraf teks dalam halaman web.</p>
    <img src="gambar.jpg" alt="Deskripsi gambar">
  </main>
  
  <footer>
    <!-- Konten footer -->
  </footer>
</body>
Header
Main Content
Footer
4

<h1> to <h6>

Elemen heading dari <h1> hingga <h6> digunakan untuk mendefinisikan judul dan subjudul dalam dokumen HTML.

<h1> adalah level heading tertinggi dan biasanya digunakan untuk judul utama halaman, sementara <h6> adalah level heading terendah. 

<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

5

<p>

Elemen <p> digunakan untuk mendefinisikan paragraf teks dalam dokumen HTML.

Browser secara otomatis menambahkan ruang (margin) sebelum dan sesudah setiap elemen paragraf untuk memisahkannya dari paragraf lain atau elemen lainnya. 

<p>Ini adalah contoh paragraf dalam HTML. Paragraf digunakan untuk mengelompokkan teks terkait menjadi satu blok.</p>

<p>Ini adalah paragraf kedua. Perhatikan bagaimana browser secara otomatis menambahkan ruang antara paragraf.</p>

Ini adalah contoh paragraf dalam HTML. Paragraf digunakan untuk mengelompokkan teks terkait menjadi satu blok.

index.html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Contoh Halaman HTML</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      line-height: 1.6;
      max-width: 800px;
      margin: 0 auto;
      padding: 20px;
    }
  </style>
</head>
<body>
  <h1>Selamat Datang di Website Saya</h1>
  
  <h2>Tentang Saya</h2>
  <p>Halo! Saya seorang pengembang web yang sedang belajar HTML.</p>
  <p>Halaman ini dibuat menggunakan elemen HTML dasar yang telah dipelajari.</p>
  
  <h3>Keterampilan Saya</h3>
  <p>Saya sedang mempelajari berbagai teknologi web, termasuk:</p>
  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
  </ul>
  
  <h4>Kontak</h4>
  <p>Silakan hubungi saya melalui email di: example@email.com</p>
  
  <h5>Tautan Berguna</h5>
  <p>Kunjungi <a href="https://www.w3schools.com">W3Schools</a> untuk belajar lebih lanjut tentang HTML.</p>
  
  <h6>Catatan Kecil</h6>
  <p>Ini adalah contoh penggunaan elemen heading level 6.</p>
</body>
</html>
